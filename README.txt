Title: Scheduling Application Manager 2.0
Purpose: GUI-based scheduling application that manages a database of appointments, contacts, customers, and users
Author: Osvaldo Ibarra
Contact: oibarr2@wgu.edu
Application Version: 2.0.1
Date: 12/31/22
IDE Version: IntelliJ IDEA Community Edition 2022.2.1
JDK Version: SDK 18 Oracle OpenJDK Version 18.0.2
JavaFX Version: SDK 19
Directions for running the program:
•	Launch the application
•	Login with a valid username and password
•	username: “test” and password: “test” will be auto populated for convenience
•	username: “admin” and password: “admin” is also valid
•	Use the Main Menu to access the Appointments table, Customer table, and the Reports Dashboard
•	Use the Add, Modify, and Delete buttons to make changes in the application
•	Use the Save button to save changes or the Cancel button to return to the Main Menu
•	Access reports by selecting the appropriate report button on the Reports Dashboard
•	Use the Exit button to exit the application when finished
Additional Report: Allows the user to filter appointments by location and provides a total count of corresponding appointments by each respective location
MySQL Connector Driver Version: 8.0.30
